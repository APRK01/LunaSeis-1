# Decision 0015: all-batch ATT and integrity policy

Date: 2026-07-13

Use the nearest valid ATT value to map catalog Earth-reception reference time to nominal MiniSEED slicing time, preserving all timing quantities. Retain the 20%/50% gap and 1 s/10 s ATT gates as primary v0.1 engineering thresholds and mandate strict/lenient sensitivity reporting. This freezes reproducibility without claiming that catalog start times are phase picks or that the absolute physical time basis is resolved.

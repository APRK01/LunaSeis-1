# Decision 0026: freeze the public release claims boundary

Date: 2026-07-14

Release LunaSeis-1 only as a reproducible research-software prototype and negative continuous-scanning study. Public material may describe the implemented 2,761-parameter depthwise model, checksum-backed Apollo pipeline, leakage controls, frozen evaluations, and development-only architecture comparison. It must explicitly preserve the two failed operational tests and the full-day persistent-activation result.

Operational, flight-ready, state-of-the-art, and absolute priority claims are prohibited. Prior HMM, CNN, and FNO Apollo studies already establish automated detection, compact neural modeling, raw-waveform use, and telemetry motivation. The defensible positioning is the combined cross-station/family-aware continuous stress test and auditable negative evidence, without claiming that this exact combination is the first in existence.
